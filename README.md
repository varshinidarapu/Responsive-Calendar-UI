🗓️ Responsive Calendar UI
A modern, interactive calendar dashboard designed for seamless planning, journaling, and productivity. This project combines smooth animations with a clean, distraction-free design to enhance your daily workflow.
🔗 Live Demo: https://responsive-calendar-ui.vercel.app/
✨ Features
3D Page-Flip Navigation: Intuitive and smooth transitions between calendar views.
Integrated Sidebar: Manage your daily journal, goals, and events in one panel.
Adaptive Theme: Full support for Light and Dark modes.
Multi-Date Selection: Easily select and manage multiple dates.
Optimized Performance: Fast interactions powered by local storage.
🛠️ Tech Stack
Framework: Next.js
Styling: Tailwind CSS
Animations: Framer Motion
Date Utilities: date-fns
🚀 Run Locally
Clone the repo
bash
git clone https://github.com/varshinidarapu/Responsive-Calendar-UI.git
Navigate to the directory
bash
cd Responsive-Calendar-UI/calendar-app
Install dependencies
bash
npm install
Start the development server
bash
npm run dev

http://localhost:3000/
