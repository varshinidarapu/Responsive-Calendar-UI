🌟 Overview

A modern, interactive calendar dashboard designed for planning, journaling, and productivity.
It combines smooth animations with a clean, distraction-free design.

✨ Features
3D page-flip calendar navigation
Daily journal + goals/events panel
Light & dark mode
Select multiple dates easily
Fast performance with local storage
🛠 Tech Stack
Next.js
Tailwind CSS
Framer Motion
date-fns
🚀 Run Locally
Clone the repo
git clone https://github.com/varshinidarapu/Responsive-Calendar-UI.git
cd Responsive-Calendar-UI/calendar-app
Install dependencies
npm install
Start the app
npm run dev
Open in browser
http://localhost:3000
